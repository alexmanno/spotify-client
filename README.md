# spotify-client
Spotify client written in bash

# Installation

```bash
wget https://raw.githubusercontent.com/alexmanno/spotify-client/master/spotify.sh
chmod +x spotify.sh
```

# Usage

```
Usage: ./spotify.sh [command]
Available commands:
        Who
        Play
        Pause
        PlayPause
        Previous
        Next
```

# Examples

## Play

```bash
./spotify.sh Play
```

## Pause

```bash
./spotify.sh Pause
```

## Next

```bash
./spotify.sh Next
```

## Previous

```bash
./spotify.sh Previous
```

## Get informations

```bash
./spotify.sh Who
```
